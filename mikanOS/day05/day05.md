# day05
## 5.1 文字を書いてみる（osbook_day05a）
- 文字を画面に表示するには、文字の部分だけを塗り、それ以外を塗らないという処理をする。
- まずは、「A」を描画するために黒く塗る部分を1、塗らない部分を0としたデータを用意する。
  #### <main.cpp（'A'のフォントデータを定義）>
  ```
  const uint8_t kFontA[16] = {
    0b00000000,   //
    0b00011000,   //   **   
    0b00011000,   //   **   
    0b00011000,   //   **   
    0b00011000,   //   **   
    0b00100100,   //  *  *  
    0b00100100,   //  *  *  
    0b00100100,   //  *  *  
    0b00100100,   //  *  *  
    0b01111110,   // ****** 
    0b01000010,   // *    * 
    0b01000010,   // *    * 
    0b01000010,   // *    * 
    0b11100111,   //***  ***
    0b00000000,   //
    0b00000000,   //
  };
  ```
